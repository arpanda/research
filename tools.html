Tools

CNVpytor

CNVpytor is a tool for CNV detection and analysis from depth-of-coverage and B-allele frequency data. It provides a comprehensive set of features for visualizing, analyzing, and interpreting CNVs.

Learn more about CNVpytor

CNVpytor Track in igv.js

The track in igv.js that utilizes CNVpytor produced pytor file and whole genome variant files as input for on-the-fly read depth and BAF visualization, CNV/CNA calling and analysis. Embedding into HTML pages and Jupiter Notebooks enables convenient remote data access and visualization simplifying interpretation and analysis of omics data.

GAL (Genome Annotator Light)

Genome Annotator Light (GAL) is a Docker-based package for genome analysis and data visualization. GAL integrates several existing tools and in-house programs inside a Docker Container for systematic analysis and visualization of genomes through a web browser. GAL takes various input types, ranging from raw Fasta files to fully annotated files, processes them through a standard annotation pipeline, and visualizes them on a web browser. Comparative genomic analysis is performed automatically within a given taxonomic class. GAL creates an interactive genome browser with clickable genomic feature tracks, a local BLAST-able database, a query page, and on-the-fly downstream data analysis using EMBOSS etc.
